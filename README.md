# 🌿 Pick Your Plants

An interactive plant encyclopedia built with HTML, CSS, and JavaScript.
Browse 16 houseplants and tap any one to learn care tips, light needs, and fun facts.

## Features
- 16 plant cards with facts and care tags
- Responsive layout — works on mobile, tablet, and desktop
- Smooth screen transitions
- Lobster Two font with a tropical aesthetic

## How to run
Just open `index.html` in any browser. No build step or dependencies needed.
